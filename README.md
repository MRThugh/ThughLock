# ThughLock

🔐 **ThughLock** is a secure and user-friendly password generator built with Python and CustomTkinter. The Windows setup installer is created using Inno Setup.

### Features
- Generate passwords using multiple algorithms: Three Words, Pattern, Diceware, etc.
- Evaluate password strength
- Keep a history of generated passwords
- Quick copy to clipboard
- Export password history to TXT

### Download
The Windows setup installer is available in the [Releases](https://github.com/MRThugh/ThughLock/releases) section.
